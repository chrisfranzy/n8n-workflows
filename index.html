🚀 Setup Instructions

Step 1: Clone the Repository

Clone the n8n-workflows repository to your local machine:

git clone https://github.com/Zie619/n8n-workflows.git

Step 2: Install Dependencies

Navigate to the repository directory and install Python dependencies:

cd n8n-workflows && pip install -r requirements.txt

Step 3: Start the API Server

Start the FastAPI backend server:

python run.py

The API will be available at http://localhost:8000

Step 4: Connect to Live Data

Once the API is running, click "Connect to API" below to load real workflow data.
